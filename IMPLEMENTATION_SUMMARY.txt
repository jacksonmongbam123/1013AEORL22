🔐 AUTHENTICATION FLOW
─────────────────────────────────────────────────────────────────────────────

1. Admin visits /adminlogin
2. Enters username (email) and password
3. POST /adminlogin → Credentials validated with Passport.js
4. OTP generated (6 random digits)
5. OTP sent to admin's email via Nodemailer
6. Admin redirected to /admin-otp page
7. Admin enters OTP
8. POST /admin-otp → OTP validated
9. Session created with adminToken
10. Admin redirected to /overview dashboard

⚙️ CONFIGURATION REQUIRED
─────────────────────────────────────────────────────────────────────────────

1. Create .env file in project root:
   cp .env.example .env

2. Add email credentials:
   EMAIL_USER=your-email@gmail.com
   EMAIL_PASS=your-app-password

3. For Gmail:
   - Enable 2-Step Verification
   - Generate App Password
   - Use App Password (not regular password)

4. For other providers:
   - Update service in Nodemailer config
   - Use appropriate credentials
